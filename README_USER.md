README - Guía de usuario

Resumen

- Esta aplicación permite gestionar proyectos inmobiliarios, comprar lotes, registrar pagos y gestionar PQR (preguntas, quejas y reclamos) a través de una interfaz web sencilla.

Acceso a la aplicación

- Abre la URL donde está desplegada la aplicación o, en desarrollo, la dirección local indicada por el equipo.

Registro e inicio de sesión

- Registrarse: accede a la página de registro y completa los datos solicitados. Después de enviar el formulario, tu cuenta quedará creada.
- Iniciar sesión: en la página de inicio de sesión introduce tus credenciales y accede al dashboard.

Dashboard

- Es la vista principal donde encontrarás accesos a proyectos, pagos, PQR y tu perfil.
- Algunos botones y opciones (crear proyectos, editar proyectos, gestionar PQR) solo se muestran a usuarios con rol de administrador.

Crear y editar proyectos (administradores)

- Los administradores pueden crear nuevos proyectos desde la opción correspondiente en el dashboard.
- Para editar un proyecto, selecciona el proyecto y utiliza la opción de editar.

Comprar un proyecto o lote

- Desde la lista de proyectos, selecciona el que te interese y utiliza la opción de compra.
- Durante la compra se solicitan datos como número de cuotas, aporte inicial y datos de contacto. Tras completar la compra, el sistema registra tu lote y genera los pagos iniciales según la configuración.

Registrar pagos

- Los pagos se registran en la sección de pagos: selecciona el lote, ingresa el monto abonado, la fecha y, si lo deseas, adjunta el comprobante.
- El sistema actualiza el estado del saldo pendiente y mantiene un historial de pagos.

Ver tus pagos y proyectos

- En la sección de pagos puedes consultar el historial de pagos realizados; desde ahí es posible obtener o imprimir un comprobante de pago.
- En la sección de proyectos/lotes verás los proyectos disponibles y los que ya has adquirido, con su estado y detalles.

Crear una PQR (usuarios)

- Si necesitas reportar una duda, queja o reclamo, utiliza el formulario de PQR disponible en la aplicación.
- Cada PQR queda registrada con un estado inicial y puede ser revisada por el equipo de soporte o por administradores.

Gestionar PQR (administradores)

- Los administradores disponen de una sección para ver todas las PQRs, actualizar su estado y añadir una respuesta.
- Al gestionar una PQR se puede marcar como resuelta y escribir una respuesta que quedará registrada para el usuario.

Consejos y problemas frecuentes

- Si no ves opciones administrativas, verifica que hayas iniciado sesión con un usuario administrador.
- Si una acción no se guarda, revisa que la conexión al servidor esté activa y contacta con el administrador si persiste el problema.

Soporte

- Para incidencias técnicas o dudas de uso, contacta con la persona o equipo responsable de la aplicación.

Observación

- Esta guía está pensada para usuarios finales y administradores de la aplicación. Si necesitas una guía técnica para despliegue o migraciones, puedo crear un documento separado con esos pasos.
